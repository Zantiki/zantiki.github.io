class src.models.models.CNN(features: int, points_in_lookback: int)¶

Bases: torch.nn.modules.module.Module

Two layered convolutional network that scales in accordance to specified input-features.

Variables

  • lookback – the size of the lookback, i.e length of each x.

  • conv – First convolutional layer, splits features into 64 sets and has a kernel-size of 5.

  • pool – First max-pool, kernel-size of 1

  • conv2 – Second convolutional layer, 64 in-channels, 128 out with kernel size 5.

  • pool2 – Second max-pool with kernel-size 1.

  • dense – Dense-layer for determining output.

  • criterion – Torch MSELoss object.
